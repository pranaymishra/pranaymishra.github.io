Syllabus

Scope – Release version Tableau 2020.3 (whichever version is latest)

General concepts

  • About Data visualization and Tableau product line
  • Review of fundamental data analysis, statistical and mathematical concepts

Filtering, Sorting & Grouping

  • Filters on Dimensions, Measures
  • Wild Cards, filters at source
  • Global, context and quick filters

Working with Data

  • Data Types and Roles – dimensions, measures
  • Database joins and Data blending
  • Working with Big Data, OLAP considerations
  • Working with the Data Engine, Custom SQL

Calculated Data and Statistics  

  • Arithmetic, string, date, table, custom and other calculations
  • Complex calculations, discrete aggregations
  • Statistical calculations, cohort analysis
  • Working with Parameters to modify title, create What-If analysis

Advanced topics

  • Charts with Real Time Applications
  • Building Interactive Dashboards  
  • Sharing Workbooks
  • Getting Started with Tableau Prep
